BOOTSTRAP
Bootstrap adalah sebuah library framework CSS yang telah dibuat khusus untuk mengembangkan front end sebuah website. 
Bootstrap juga dikenal sebagai salah satu framework CSS, HTML, Javascript yang begitu populer di kalangan website 
developer atau pengembang website.Fungsi dari bootstrap yaitu untuk mengembangkan website agar lebih responsive.

di website saya,saya menggunakan bootstrap untuk membuat navigation bar.Keuntungan dari menggunakan bootstrap
saya tidak perlu lama mengerjakan navbar karena sudah ada templatenya yang termasuk membuat navbarnya responsive.
Di bootstrap untuk membuat navbar yang responsive sendiri memerlukan bahasa html css dan javascript dengan line
code yang sangat panjang. Sehingga dnegan menggunakan bootstrap saya dapat menghemat waktu dalam pembuatan 
website.
